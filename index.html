Fonctionnement de flexbox

Flexbox fonctionne avec :

Un flex-container qui permet de créer le contexte pour l'utilisation de Flexbox. Le ou les enfants directs d'un élément avec la propriété display: flex; deviennent automatiquement des flex-items. Il est possible d'appliquer la propriété display: flex; ou display: inline-flex; à tous les types d'éléments HTML.

Propriétés pour le parent (flex container)

display

On donne la propriété display:flex; à un élément pour lui donner un contexte flex qui va permettre à ses enfants directs d'être flexibles

1
2
3
code
.container {
    display: flex; /* ou inline-flex */
}

À savoir : à partir du moment ou un élément HTML a la propriété display: flex, certaines propriétés n'auront plus d'effet sur lui. La propriété columns de CSS ne fonctionne plus correctement, ainsi que float.

flex-direction

Cette propriété permet de déterminer le sens d’affichage sur l'axe horizontal ou l'axe vertical des éléments flex-items (enfants) d'un flex container (parent).

Ses différentes valeurs sont :

  • row (ou rangée. C'est la valeur par défaut des flex-items. Cela donne l'alignement horizontal)
  • row-reverse (inverse le sens des éléments alignés horizontalement
  • column (alignement vertical. Il s'agit de l'alignement original des éléments HTML qui ont la propriété display: block; par défaut.
  • column-reverse (inverse le sens des éléments alignés verticalement

Propriétés pour les enfants (flex items)

order

Par défaut, les flex items s'affichent dans le même ordre que la source (le fichier HTML). Cependant, la propriété order permet de contrôler l'odre des flex-items dans un flex container.

order: 1

1

2

3

4

5

6

7

code
.item {
       order: un nombre entier;
    }

flex-grow

Cela permet d'élargir un flex item si on en a besoin. On lui donne une valeur sans unité de mesure qui correspond a une proportion par rapport aux autres éléments (flex items). Ces unités déterminent l'espace que prendra l'élément en fonction de l'espace disponigle de le flex container.

Si tous les éléments on une valeur flex-grow: 1; ils prendront tous la même quantité de place dans l'espace disponible dans le flex container. Si un des flex-items a une valeur de 2, il prendra 2 fois plus de place que les autres éléments avec une valeur 1.

flex-grow: exemple 1

1

1

1

code
.item {
       flex-grow: 1;
    }
flex-grow: exemple 2

1

2

1

code
.flex-grow-2 .item {
       flex-grow: 1;
    }
    .flex-grow-2 .item:nth-child(2) {
       flex-grow: 2;
    }
flex-grow: exemple 3

1

2

3

code
.flex-grow-2 .item {
       flex-grow: 1;
    }
    .flex-grow-2 .item:nth-child(2) {
       flex-grow: 2;
    }
    .flex-grow-3 .item:nth-child(3) {
       flex-grow: 3;
    }

flex-shrink

Cela permet de définir la capacité d'un élément flex à rétrécir si nécessaire.

Il n'est pas possible d'utiliser des nombres négatifs.

flex-shrink: exemple 3

1

2

3

code
.item {
       flex-shrink: nombre;
    }

flex-basis

Cela permet de définir la taille par défaut d'un élément avant que l'espace qui l'entoure soit redistribué. Cela peut-être une longueur (exemple : 20%, 6rem, etc.) ou bien cela peut-être un mot clé. (les mots sont clés sont encore très mal supportés par les navigateurs, nous éviterons donc de les aborder pour le moment.

Si il est réglé à 0, l'espace distribué autour du contenu n'est pas pris en compte. Si il est réglé sur auto, l'espace disponible est redistribué en fonction de la valeur flex-grow de l'élément.

code
.item {
      flex-basis: longueur | auto; /* default auto */
    }

flex

C'est la version raccourcie de flex-grow, flex-shrink et flex-basis. Le second et troisième paramètres (flex-shrink et flex-basis) sont optionnels. Les valeurs par défaut sont : 0 1 auto.

Il est recommandé d'utiliser cette forme plutôt que de spécifier les propriétés individuellement. La version raccourcie rèble automatiquement les autres valeurs intelligement.

code
.item {
      flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
    }

align-self

Cela permet d'écraser l'alignement précisé par la propriété align-items pour des éléments flex individuels. Voir la propriété align-items pour comprendre quelles sont les valeurs possibles.

code
.item {
      align-self: auto | flex-start | flex-end | center | baseline | stretch;
    }

Il est important de noter que les propriétés float, clear et vertical-align n'ont pas d'effet sur un élément flex.
