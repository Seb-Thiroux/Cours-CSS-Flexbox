Layout 3 colonnes.

Comment est construite cette page ?

La mise en page est réalisée grâce à Flexbox. Il y a entre autres un menu flexible qui change de forme en fonction de media queries défini dans CSS. Il y a également un footer qui reste en bas de page (sticky footer) et enfin la distribution de différentes parties de la page s'adapte elles aussi en fonction des media queries

La partie contenant le contenu principal du site main et les deux aside sont enveloppées dans un contenant auquel on a donné la classe .wrapper. Ce dernier permet de créer le contexte flex avec la propriété display: flex;. On ajoute la propriété flex-flow: column wrap afin de répartir le contenu en une seule colonne pour les petits écrans (les blocs s'empiles.

Au passage du premier media querie, on change la propriété flex-flow: row wrap pour obtenir des éléments qui s'alignent horizontalement. On ajoute la propriété flex: 1; aux éléments avec la classe .aside pour leur dire de prendre l'espace dont ils ont besoin sur la rangée, ce qui permet de les aligner côte à côte. Avec le troisième media querie, on donne la propriété flex: 3; au contenu principal .main, ce qui va l'intégrer dans le contexte flex-flow: row;. En lui donnant la valeur 3 on indique qu'il doit prendre 3 fois plus d'espace que les autres éléments avec une valeur flex: 1;.

Les éléments HTML sont ensuite ré-ordonnés avec la propriété order: un numbre;

code 
.wrapper {
   display: flex;
   flex-flow: column wrap;
   flex: 1;
}

Vous noterez dans le code ci-dessus qu'on a également donné la propriété flex: 1; à l'élément avec la classe wrapper. C'est parce que celui-ci se trouve également dans un contexte flex défini sur l'élément body. Ceci nous permet d'avoir un footer qui reste en bas de page même si celle-ci est vide.
